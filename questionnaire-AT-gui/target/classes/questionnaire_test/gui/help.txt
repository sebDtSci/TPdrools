Questionnaire
... 	: Choisir le fichier pour charger / sauvegarder le questionnaire.
CTRL + s 	: Sauvegarder le questionnaire en l'état dans le fichier.
CTRL + o 	: Ouvrir le questionnaire depuis le fichier.

Contexte
... 	: Choisir le fichier pour charger / sauvegarder le contexte 
CTRL + s 	: Sauvegarder le contexte dans le fichier. Le contexte est sauvegardé et rechargé dans l'application.
CTRL + o 	: Ouvrir le contexte depuis le fichier.
CTRL + l 	: Charger le contexte depuis la zone de texte.
